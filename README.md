# Emotion-Based Movie Recommendation System

A Flask-based web application that recommends movies based on emotions
detected from user text input using Natural Language Processing (NLP).

---

 Technologies Used
- Python
- Flask
- Natural Language Processing (NLP)
- Pandas, NumPy
- HTML, CSS
- TMDB Dataset

---
 Features
- Emotion detection from user input text
- Supports emotions such as happiness, sadness, anger, and fear
- Recommends movies using similarity-based filtering
- Dynamic web interface using Flask templates

---
## Note
Due to GitHub file size limitations, large dataset and model files
are not included in this repository. These files are generated
during preprocessing and can be recreated when running the project locally.
